**Текущий коммит:** `4be0514`  
**Среда проверки:** Локальная сборка Next.js 16.2.9 (Turbopack) / Production Build

## 1. Техническое SEO (Technical SEO Foundation)

Реализованы основные требования для эффективного сканирования и индексирования сайта поисковыми системами:

1. **Динамический файл `robots.txt` (`src/app/robots.ts`):**
   * Закрыты от индексации все служебные разделы, личные кабинеты и API-маршруты:
     * `/crm`
     * `/login`
     * `/teacher`
     * `/parent`
     * `/student`
     * `/api/*`
   * Разрешено индексирование главной страницы и всех публичных целевых посадочных страниц.

2. **Динамическая карта сайта `sitemap.xml` (`src/app/sitemap.ts`):**
   * Содержит только публичные индексируемые страницы:
     * Главная (`/`)
     * Пробное занятие (`/probnoe-zanyatie`)
     * Расписание (`/raspisanie`)
     * Стоимость (`/stoimost`)
     * Политика конфиденциальности (`/privacy-policy`)
     * Согласие на ОПД (`/consent`)
     * Страницы учебных направлений (робототехника, Scratch, Python, Arduino и общая страница программирования).

3. **Метаданные страниц и заголовки:**
   * На главной странице настроены: `metadataBase` (`https://robotics-lipetsk.ru`), canonical, Open Graph и Twitter Card метатеги.
   * На всех закрытых разделах (`layout.tsx` для `/crm`, `/login`, `/teacher`, `/parent`, `/student`) принудительно установлены метаданные:
     ```typescript
     export const metadata = {
       robots: {
         index: false,
         follow: false,
       },
     };
     ```
   * Все публичные страницы содержат уникальные метаописания и канонические ссылки (`canonical`).

## 2. Разделение Landing Page (Server-Side Rendering)

Для поддержки генерации метатегов и поисковой микроразметки на стороне сервера главная страница была разделена:
* **`src/app/(public)/page.tsx`** — серверный компонент. Отвечает за генерацию SEO-метаданных и внедрение разметки JSON-LD.
* **`src/app/(public)/LandingPageClient.tsx`** — клиентский компонент. Содержит интерактивные блоки, форму захвата лидов, анимации и диалоги робота-помощника.

---

## 3. Разметка структурированных данных (JSON-LD Schemas)

Внедрена микроразметка для улучшения представления сайта в результатах поиска (Rich Snippets):
* **`EducationalOrganization` / `LocalBusiness`** на главной странице (содержит адрес кружка в Липецке, телефон, часы работы и гео-координаты).
* **`Course`** на страницах учебных направлений (робототехника, программирование, Scratch, Python, Arduino) для правильного отображения курсов в поиске Google/Яндекс.
* **`BreadcrumbList`** на всех внутренних страницах для навигационных цепочек.

## 4. Конверсионные улучшения и контент

Реализованы доработки интерфейса для повышения процента конверсии в заявки на пробный урок:

1. **Hero-блок с конкретным оффером:**
   * Заголовок изменен на: *«Бесплатное пробное занятие 90 минут: ребенок соберет и запрограммирует первого робота»*.
   * Под кнопкой CTA добавлены ценностные триггеры: *Без предоплаты*, *Оборудование включено*, *Мини-группы до 8 детей*, *Подберем группу по возрасту*.

2. **Блоки доверия (Trust Building):**
   * **Проекты учеников:** Выводятся реальные проекты (автоматическая микро-теплица на Arduino и робот-сумоист на LEGO Mindstorms) с подробным описанием результатов первого месяца.
   * **Преподаватели:** Карточки с квалификацией наставников, их цитатами и уклоном.
   * **Фото классов и оборудования:** Bento-сетка с фотографиями учебного процесса и наборов LEGO Education.
   * **Контакты и Карта:** Блок с физическим адресом (г. Липецк, ул. Ленина, д. 10), часами работы и юридическими реквизитами ИП (ОГРНИП, ИНН, Лицензия на образовательную деятельность).

3. **Интерактивная форма захвата (`#lead-form`):**
   * Поддерживает полный набор полей: Имя родителя, Имя ребенка, Телефон, Возраст ребенка, Учебное направление, **Удобное время занятий** (выпадающий список) и комментарий.
   * Добавлен обязательный флажок согласия с Политикой обработки персональных данных, ссылающийся на `/privacy-policy` и `/consent`.
   * Во время отправки кнопка корректно блокируется для защиты от двойного клика.

## 5. Интеграция систем аналитики (Analytics Hooks)

* Внедрены коды отслеживания для **Yandex Metrika** (`NEXT_PUBLIC_YANDEX_METRIKA_ID`) и **Google Analytics** (`NEXT_PUBLIC_GA_ID`). Скрипты автоматически подключаются через `next/script` на публичных страницах.
* Настроены JS-события для отправки целей конверсии:
  * `lead_form_submitted` (успешная отправка заявки)
  * `phone_clicked` (клик по номеру телефона)
  * `cta_button_clicked` (клики по кнопкам записи на пробный урок)
  * `scrolled_to_lead_form` (скролл пользователя до формы заявки)
  * `course_page_viewed` (переход на страницы детального описания курсов)
